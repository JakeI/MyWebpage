Taylor Reihe¶

$$ f(x_1,\, \cdots \, ,\, x_n) = \sum_{j=0}^\infty \Bigg( \frac{1}{j!} \bigg( \sum_{k=1}^n \big(\underbrace{x_k-a_k}_{\Delta x_k}\big) \frac{\partial}{\partial x_k'} \bigg)^j f(x_1', \cdots, x_n') \Bigg) \Bigg|_{ \matrix{x_n' = a_n \\ \vdots \\ x_1' = a_1}} $$

                gridcolor='rgb(255, 255, 255)',

                showbackground=True,

            ),

                gridcolor='rgb(255, 255, 255)',
                zerolinecolor='rgb(255, 255, 255)',
                showbackground=True,
                backgroundcolor='rgb(230, 230,230)'
            )
        ),
        updatemenus=list([
            dict(
                buttons=list([   
                    dict(
                        label="Both", method="update",
                        args = [dict(visible=[True, True]),
                                dict(title="Both")]
                    ),
                    dict(
                        label="f(x,y)", method="update",

                    ),

        ])
    )

    fig = dict(data=data, layout=layout)

    off.plot(fig, filename="1b_%s.html" % text)

    disp.display(disp.HTML("""<p>
        <a style='font-size: 40pt' href='1b_%s.html' target='_blank'>
        %s</a></p>""" % (text, text)))
