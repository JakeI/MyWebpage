3D Graphics Engine using PyOpgenGL and PyQt

This is a work in progress.

Currently Supported Features include:
  • A math library.
  • Loading GLSL shades form xml formatted files
  • Loading *.obj files
  • Generally loading resources in bulk
  • Global parameter and a GUI to edit them
  • Unlimited viewports all of witch share their resources
  • A full post processing pipeline with a viewport specific editable set of parameters
  • A Python console for dynamic interaction
  • 3D mouse picking (mouse to object, mouse to location)
  • Standard forward lighting
  • Easy integration of C code
  • Skybox rendering
